# CookieClicker
Using MASM this cookie clicker was created by Jacob Woods and Santiago Raigoza

RUNNING THE GAME:
download main.asm and can run simply in visual studio code by "building solution" and then "start without debugging"

CONTROLS:
Space bar,
keys 1-6,
q

PLAYING THE GAME:
repeatedly pressing space bar will get you cookies, or points that can then be used to buy upgrades that grant faster cookie generation, reaching a 
certain value above 2 million you can buy "Win the Game".
